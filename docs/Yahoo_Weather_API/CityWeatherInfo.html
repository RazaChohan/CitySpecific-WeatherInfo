Gets the WOEID, Timezone, CityName and Type of Place from Yahoo Weather API


This method/funciton gets city name as a parameter and makes an API call  to Yahoo Weather API. The Yahoo Weather API returns in XML format. Using the  XML feed returned from Yahoo API this function gets Timezone, Country name,  WOEID(Where On Earth ID) and type of place.
